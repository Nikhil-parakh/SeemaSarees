# Seema Sarees

A production-ready, premium branding website for a saree manufacturing business built with React 19, Vite, Tailwind CSS, Framer Motion, and React Icons.

## Features

- Mobile-first responsive design
- Smooth scrolling navigation
- Animated sections with Framer Motion
- Reusable, component-based architecture
- Editable content stored in `src/data/content.js`
- Gallery with modal preview
- Inquiry form with validation and success state
- Floating WhatsApp contact button

## Installation

1. Open a terminal in the project root.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

5. Preview the production build:

```bash
npm run preview
```

## Notes

- Content is centralized in `src/data/content.js`.
- The site is designed for wholesale inquiries and brand credibility, not ecommerce.
- Replace placeholder images in `src/assets/` with brand photography when available.
